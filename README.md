# PhishTest
Python script that validates if a URL is malicious. 
Current Version: 1.0

# Requirements
Python 2.7.x

Libraries: Json, SimpleJson, Urlib, Urlib2

Application Keys:
* VirusTotal
* PhishTank

# How-To
It's as simple as 1, 2, 3... 

1. Start up the script by just calling it or double clicking it straight from the directory
2. Enter your URL and wait for results
3. Enter Y/y to scan anothher URL or N/n to exit
